Parcel Game Engine

3D game engine developed in C++

Parcel Game Engine's main goal is to abstract and simplify the lower level details of games (e.g. graphics rendering and resource management) away from the user, so they may focus on developing the actual game instead. It is primary aimed towards aiding the development of quick prototypes, but can be used for full-fledged games as well. Features include:

  • Hardware accelerated 2D graphics
  • Hardware accelerated 3D graphics
  • Texture loading and configuration
  • Vertex/fragment/pixel shaders
  • Multiple camera types
  • Built-in geometric primitives
  • Automatic resource management
  • Lua scripting engine
  • OGG and WAV loading for sound
  • OBJ and DMF loading for meshes
  • TTF loading for text
  • Tools for debugging and logging

All development on this project ended in August 2010, since I did not have time to maintain the project. However, if you're curious, head to the project page to view the source code, download the engine and play around with it.

IATI Viewer

Interactive cash flow and activity browser for charities and NGOs

I entered J.P. Morgan’s Code for Good 2012 challenge as part a team of four software developers. We created a prototype web application in two days for the charity WaterAid. There were two aims. The first was to allow charities to easily explore the large amounts of monetary transaction and activity data they had. The second aim was to use the transparency of monetary flow and activities this tool provides to promote charities to the general public.

The result was a web visualisation tool which allows interactive exploration of activity and financial data for various charities and non-governmental organisations. Activities and financial transactions are represented as nodes on a graph. These graphs can become very large, so the user is first given a high-level view of the data. Nodes in the graph can then be expanded to drill down and see greater detail on specific activities/transactions.

Anonybew

Chat client that provides confidential helplines for university students

For most students their time at university is a positive experience, however for some it can be characterised by periods of stress, anxiety, loneliness and depression. Nightline Association gives students safe, anonymous places students can use to talk about their problems.

A friend who volunteered for Nightline got in touch with me about developing an anonymised web chat client for University of Warwick's nightline website. The application could not persistently store any messages or keep any information about the students who used the service.

I created an anonymised version of Mibew, an existing open-source, web-based messaging application. I changed the internals of the system so messages were not persistently stored and no client information was shown to the operators.

Go to the University of Warwick's nightline website to see Anonybew in action. To download Anonybew or view the source code, check out the project page.

mdsearch

C++ library which implements several high performance, multi-dimensional search structures

Many fields require or benefit from multi-dimensional search, including database querying, machine learning, computer vision and visualisation. In multi-dimensional search, individual records are represented as points, vectors or regions in n-dimensional space. These are arranged in an index structure so relevant data can be retrieved quickly, even when the volume of data is huge.

Originally developed as part of my final year project at university, mdsearch is an open-source library that implements several multi-dimensional search structures. Its sole aim is search through multi-dimensional datasets as quickly as possible.

Pre-existing multi-dimensional search structures have been implemented in the library, such as the kd-tree and pyramid tree. Two novel structures have been also added to the library ‐‐ specifically, the Bit Hash and Multigrid Tree structures.

Source code and documentation is available in the project's GitHub repository.

YourView

Analyse population's opinion and sentiment towards major world issues

YourView is a prototype system developed by a team of five developers at HackaSoton. Sentiment analysis techniques are used to determine how people feel about certain topics that have deemed "current affairs" by the system. For example, if there is currently an ongoing conflict between two countries, YourView will pick that up. It'll then start analysing the public's opinion about the conflict using social media data and store this information. The information is then retrieved and displayed to users via a web interface.

YourView is composed of two parts ‐‐ the back-end system which does the sentiment analysis and the front-end which gives users a way of exploring current world affairs. The back-end continuously receives social media data and analyses it to give an up-to-date view of the public's opinions on the front-end. How it works:

Back-End

  1. Social media posts (e.g. Facebook, Twitter, etc.) are pushed to the Sentiment Analyser
  2. Sentiment Analyser determines what topics the post is about and whether the poster is talking positively or negatively about them
  3. Sentiment Analyser updates stored sentiment data of new post's topics in the Opinion Database

Front-End

  1. Users are presented with topics that may interest them
  2. The public's sentiments/opinions on the user's chosen topics, taken from the Opinion Database, are presented to users

In addition to current, global affairs, YourView can analyse public sentiment of any kind of topic, such as an organisation, TV show, product or even a general concept (e.g. socialism). All the system needs is some information about the topic to analyse (e.g. keywords and textual documents about it) and it'll do the rest.
